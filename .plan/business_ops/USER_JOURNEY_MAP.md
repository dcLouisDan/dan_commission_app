# User Journey Map

This document maps the emotional and functional experience of a client commissioning artwork. The goal is to minimize friction and anxiety.

| Phase | Client Action | Artist Action | Key Touchpoints | Emotional State |
| :--- | :--- | :--- | :--- | :--- |
| **1. Discovery** | Browses Portfolio / Twitter / Instagram. Likes art style. Checks prices. | Maintains up-to-date portfolio and price sheet. | Social Media, Portfolio Site | *Intereseted, Cautious* |
| **2. Decision** | Decides to commission. Looks for "How to Order" or "TOS". | Provides clear "Commission Open" status and links to TOS. | Carrd / Website / pinned post | *Excited, Calculating budget* |
| **3. Onboarding** | **Fills out Commission Form.** <br> *Inputs: Ref sheets, Character Name, Personality, Pose preference, Email.* | Receives notification. Reviews request for feasibility. | Google Form / Website Form | *Hopeful (Will they accept?)* |
| **4. Agreement** | Receives acceptance email + Invoice. Pays deposit. | Sends professional acceptance email with clear quote and payment link. | Email / Invoicing Platform | *Committed, Trusting* |
| **5. Production (Sketch)* | Waits for update. Receives sketch. Provides feedback. | Draws rough sketch. Emails "Sketch for Review" template. | Email / Discord | *Anxious -> Relieved (Seeing progress)* |
| **6. Production (Final)* | Approves sketch. Waits for final render. | Finishes artwork. | - | *Anticipating* |
| **7. Delivery** | Receives final files. writes review/thank you. | Sends "Final Delivery" email with GDrive link. Asks for feedback. | Email | *Delighted / Satisfied* |

## Onboarding Optimization (The Form)
To reduce back-and-forth emails, the intake form MUST require:

1.  **Project Type**: (Headshot / Half-body / Full-body)
2.  **Character Refs**: (Link to Google Drive folder or image upload - NO text descriptions for visual traits if possible)
3.  **Personality/Vibe**: (e.g., "Cheery", "Gloomy", "Aggressive") - Helps with expression.
4.  **Specific Preferences**: (e.g., "Make sure the scar is on the LEFT eye")
5.  **Email Address**: For sending the Xendit invoice.
6.  **Agreement Checkbox**: "I have read and agree to the Terms of Service."
